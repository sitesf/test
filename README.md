# NEXAS CORE CITY

Proiect static 3D pentru un landing page cinematic cu agenti AI.

## Cum il rulezi

1. Deschide `index.html` in browser.
2. Ai nevoie de internet, pentru ca Three.js se incarca din CDN.
3. Fa scroll lent. Camera trece prin oras.

## Ce include

- Oras 3D procedural
- Strazi si cladiri neon
- Roboti AI
- Robot ghid
- Oameni, masini si drone
- Ploaie de CV-uri
- Telefon WhatsApp urias
- Fabrica PDF / Excel
- Turn Analytics
- NEXAS CORE
- Particule de date
- Portaluri
- HUD cu statistici
- Text cinematic pe scroll

## Limitari

Nu foloseste modele 3D externe `.glb`.
Robotii si orasul sunt generate direct din cod Three.js.
Pentru un nivel tip joc video real, urmatorul pas este sa adaugi modele facute in Blender.
